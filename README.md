# go-dropbox

[Dropbox SDK for Go \[UNOFFICIAL\]](https://github.com/dropbox/dropbox-sdk-go-unofficial)でDropboxのAPI叩いてみる。

## インストール

```sh
❯ go get github.com/dropbox/dropbox-sdk-go-unofficial/dropbox/...
```
